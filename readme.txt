== Description ==
See the <a href="https://www.ringroost.com/click2call.php">Demo Click To Call Button Here.</a>

This plugin work on modern browsers, and has fall back if browser does not support WebRTC see support here: <a href="http://iswebrtcreadyyet.com/"> http://iswebrtcreadyyet.com/ </a> 

Note: HTTPS is required for click to call button to work on Chrome.  

https://www.youtube.com/watch?v=pVpl6LF-xxg

WP Call Me is the “go to” system for adding a desktop click to call button on your website and managing calls that come into your website.  WP Call Me allows you to setup click to call functionality on your website, call flow systems , call queues , interactive voice response systems and more - all using a simple drag and drop interface. You can set up automated voice response systems, capture and check voicemails, place calls,  setup phone menu system, filter out callers and much more.


PLEASE NOTE: "Click to call" calls are not free, but billed on a per minute bases find <a href="https://www.ringroost.com/pricing.php">pricing here </a>.  WP Phone works by leveraging and integrating   <a href="https://www.ringroost.com"> RingRoost’s </a> system which allows users to build out call flow systems. You will need to register an account with ivrdesigner, which can be done within this plugin. 





<h4>Common Use Cases:</h4>

* Click to call buttons on your website 
* Businesses looking to setup an interactive voice response menu, where the caller can choose what number or department they want to be redirected to. 
* Businesses who want to capture input from users via phone. 
* Many other uses -  just get creative...

<h4> Requirements: </h4>

== Installation ==

1. Upload `WP Call Me` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Register with RingRoost
4. Embed the call me button using the shortcode.

== Frequently asked questions ==

= How do I use WP Call Me? =
Simply Install the plugin, register an account and add the shortcode to your page

= What does the default call system do? =
1. Click to call button, allows users to call from their browser 
2. Interactive voice response systems
3. Inbound call flow


= How much do calls cost?  =
All calls go through www.ringroost.com so they will set the prices.
